# 🚀 KidPedia

KidPedia est une encyclopédie interactive conçue pour les enfants, bâtie avec React, TypeScript et Zustand. L'application propose une expérience d'apprentissage ludique à travers des thèmes variés, un jeu de plateau "Safari" et une exploration historique "Origins".

## 📚 Documentation du Projet

La documentation technique et fonctionnelle est centralisée dans le dossier `/docs` :

- **[Guide & Référence Technique](./docs/GUIDE.md)** : Vision, règles de développement, architecture et état de la dette.
- **[Historique des Versions](./docs/HISTORY.md)** : Suivi complet de l'évolution du projet (Changelog).

## 🛠️ Installation et Lancement

1.  **Installer les dépendances** : `npm install`
2.  **Lancer en mode développement** : `npm run dev`
3.  **Valider le code (Lint + Types + Tests)** : `npm run validate`
4.  **Lancer les tests E2E** : `npx playwright test`

## 🏗️ Architecture

L'application utilise une architecture moderne basée sur :
- **Vite** pour le build ultra-rapide.
- **Zustand** pour une gestion d'état atomique et persistante.
- **CSS Modules** pour un styling encapsulé et propre.
- **Vitest & Playwright** pour une robustesse garantie.
